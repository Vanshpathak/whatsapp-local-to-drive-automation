This python automation code enables the user to transfer specific file types to specific location.
Usage: python
Libraries used: tkinter , Shutil, os, time

This repository contains my python code which changed to .exe file by pyinstaller for background running at startup without any terminal which manages and sorts the specific type of files (with specific extensions pdf,docx,pptx,xlsx)to a particular google drive folder from whatsapp temporary location , and later displays in overlay that the particular file is moved. This enables automation of files to a specific folder.
